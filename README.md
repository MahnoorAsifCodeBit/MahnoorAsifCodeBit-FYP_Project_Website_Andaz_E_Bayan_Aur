**Andaz-e-Bayan Aur** is an innovative web-based platform designed to revive and modernize the experience of **Urdu poetry and calligraphy** for the digital age. The project primarily targets **Generation Z**, blending rich literary heritage with modern technologies such as **AI, Augmented Reality (AR), and interactive web design**.

This platform offers a culturally rooted yet technologically forward experience, enabling users to explore, customize, and interact with Urdu poetry in engaging and meaningful ways.
